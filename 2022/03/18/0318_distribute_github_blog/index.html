깃허브 블로그(hexo)

ex) 는 형식에 맞춰서 재작성, 없으면 그대로 적으면 됨.

필수 파일 설치

  • nodejs.org 설치 (16.14.1 LTS 사용)

  • git-scm.com 다운로드

  • hexo 설치 (npm을 통해서 설치 가능)

깃허브 설정

  • Repositories (이하 Repo)는 백업용 + 배포용 두가지를 권장

  • 백업용 Repo를 git clone을 통해 적당한 경로로 내려받는다.

블로그 만들기

  • 폴더 만들기

  • 폴더의 Hexo를 초기화 후 설치

  • _config.yml 파일 설정

    • 사이트 정보 수정
    • 블로그 url 설정
    • 깃허브 연동
